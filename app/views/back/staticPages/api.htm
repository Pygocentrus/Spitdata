Intro

Ready to start ? Okay, now, all you need in order to call our API is a cross-domain AJAX request. You can do it with the native XmlHttpRequest Object from your favorite browser, or by a javascript framework. As an example, here's the way you can call our API using JQuery framework :

Tip : Don't forget to set the datatype to jsonp, allowing cross-domain calls

					$.ajax({
						url: http://spitdata.com/user/10,
						datatype: "jsonp"
					}).done(function(data) {
					    if (data) {
					      console.log("Alright buddy, you got your data !");
					    }
					});

Tweets

Images

Obtain images choosing the theme between articles illustration (articles), profiles pictures (profile), . Then choose the width width (in px) and height (in px) you want. Add the extension between png, jpg and gif and the number of images you want. If you ask for one image, the image will appear in a window (content-type: image/@extension). If you ask for many images you will obtain some JSON. 
http://spitdata.com/image/@theme/@width/@height/@extension/@nbImages/@zip
One image : 
http://www.spitdata.com/image/articles/400/500/gif/1
Several images : 
http://www.spitdata.com/image/articles/400/500/gif/6
If you want to obtain your images in a zip file, just add ‘zip’ at the end of you url. 
http://www.spitdata.com/image/articles/400/500/gif/6/zip
