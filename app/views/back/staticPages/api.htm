Intro

Ready to start ? Okay, now, all you need in order to call our API is a cross-domain AJAX request. You can do it with the native XmlHttpRequest Object from your favorite browser, or by a javascript framework. As an example, here's the way you can call our API using JQuery framework :

Tip : Don't forget to set the datatype to jsonp, allowing cross-domain calls

					$.ajax({
						url: http://spitdata.com/user/10,
						datatype: "jsonp"
					}).done(function(data) {
					    if (data) {
					      console.log("Alright buddy, you got your data !");
					    }
					});
